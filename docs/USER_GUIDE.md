# 用户使用手册

欢迎使用 AI-Buddy。

这是一个**先让你 5 秒记下来，再慢慢串成线**的工作空间——任务、备忘、阅读、随记四种容器共用同一套关联机制，碎片化输入与结构化整理是同一件事的两个阶段。

本文档介绍所有功能的使用方法。

## 第一次使用

打开网站后，会跳转到登录页。点击 **「立即注册」** 创建你的账号：

- **用户名**：3-30 个字符，只能包含字母、数字、下划线
- **昵称**（可选）：留空则使用用户名作为显示名
- **密码**：至少 6 个字符

注册成功后会自动登录，并为你创建 7 个预设任务分组（品牌发展、营运标准、加盟商管、产运数据、日常管理、个人项目、其他）。

## 用 AI-Buddy 做事的两种方式

**直接操作**：浏览器里打开 AI-Buddy 页面，自己建任务、记备忘、收藏阅读、丢随记。

**让 AI 代劳**：登录网页 → 个人设置 → API Key Tab → 创建一个 Key → 然后在 Claude / GPT 等 AI 助手中加载 `buddy-skill/` 目录。AI 就可以帮你查任务、加备忘、整理、归档——删除和整理前会自动列计划等你确认。详见 [buddy-skill/SKILL.md](../buddy-skill/SKILL.md)。

## 顶部导航

### 桌面端

顶部包含：Logo、当前页面标题、日期与周数、用户头像与昵称、登出按钮。

### 移动端

顶部包含：标题、日期、登出按钮。
底部有 5 个 Tab：总览、任务、备忘、阅读、梳理。

## 总览（Dashboard）

打开应用默认显示的页面，集中展示：

- **今日任务**：今日到期的任务数
- **总任务数**：所有未完成任务的统计
- **任务分组概览**：每个分组下的任务数量
- **最近备忘**：最近 5 条备忘录
- **最近阅读**：最近 5 条阅读收藏
- **随记数量**：累计记录了多少条随记

## 任务（Tasks）

任务功能是本应用的核心。

### 任务分组

左侧栏是分组列表，每个分组用不同颜色标识。点击分组可筛选该分组下的任务。

每个任务包含：

- **标题**
- **描述**（富文本，支持 Markdown）
- **状态**：待办、进行中、已完成、已归档
- **优先级**：高、中、低
- **子任务**：一个任务可拆分为多个子任务
- **进度**：0-100%
- **截止日期 / 计划日期**
- **负责人 / 支持人**
- **关联备忘 / 关联阅读 / 关联文档**
- **依赖关系**：前置任务与后续任务

### 创建任务

点击 **「+ 新建任务」** 按钮，填写表单保存。任务 ID 由系统自动生成。

### 任务依赖

在任务详情中可以设置前置任务（必须先完成才能开始本任务）和后续任务（本任务完成后才能开始的任务）。这在「梳理」视图中会显示为流程图。

## 备忘（Memos）

备忘用于记录需要长期保存的笔记、想法、文章摘录。

### 创建备忘

点击 **「+ 新建备忘」**，填写：

- **标题**（可选）
- **内容**（富文本）
- **类型**：笔记 / 想法 / 文章
- **标签**：多个标签用逗号分隔
- **关联任务 / 关联阅读**：与具体任务或文章建立联系

### 查找备忘

顶部搜索框支持按标题和内容模糊搜索。也可以按标签筛选。

## 阅读（Reading）

收藏有价值的文章链接。

### 添加阅读

点击 **「+ 添加阅读」**，粘贴 URL，系统会尝试抓取标题。如果抓取失败可手动填写。

每条阅读可以：

- 标记为已读 / 未读
- 收藏为星标
- 添加标签
- 关联备忘或任务

## 随记（Quick Notes）

最轻量的记录方式，适合会议中、走路时突然冒出的想法。

在 **任务** 页面底部或 **总览** 页面有快速输入框，键入内容回车即可保存。点击随记可编辑或删除。

## 梳理（Note View）

梳理功能是本应用的特色。当你有多个相关的任务、备忘、阅读需要集中整理时使用。

### 打开梳理

点击任务详情中的「梳理」按钮，或顶部导航的「梳理」Tab。

### 使用梳理

- 左侧是任务列表
- 中间是富文本编辑区
- 右侧是关联的备忘和阅读
- 可以拖拽调整窗口大小
- 窗口可最小化、关闭

适合场景：

- 准备周报
- 整理一个项目的所有相关资料
- 写一篇主题文章

## 移动端使用

- 顶部 11 号高顶栏：日期、登出
- 底部 5 个 Tab：总览、任务、备忘、阅读、梳理
- 任务和备忘页面：单列展示
- 任务卡片：点击进入详情

## 登出

点击顶部右上角的登出图标（PC 端在头像旁，移动端在右上角），即可退出登录。

## 常见问题

**Q: 我的数据安全吗？**
A: 数据存储在你自己的服务器上，只有你能访问。后端使用 JWT 认证，密码 bcrypt 加密。

**Q: 忘记密码怎么办？**
A: 当前版本暂未提供密码重置功能。如需重置，可在服务器执行：
```bash
mysql -u buddy -p'你的密码' buddy -e "
  UPDATE users SET password_hash = '<新的bcrypt哈希>' WHERE username = '你的用户名';
"
```
生成新的 bcrypt 哈希：<https://bcrypt-generator.com/>

**Q: 怎么备份我的数据？**
A: 备份 MySQL 数据库即可：
```bash
mysqldump -u buddy -p'你的密码' buddy > backup_$(date +%Y%m%d).sql
```

**Q: 可以多人协作吗？**
A: 当前版本每个用户的数据完全独立，不支持共享任务池。团队协作功能在路线图中。
